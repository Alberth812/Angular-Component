# Angular-Component
Proyecto: Aplicación Angular Material Demo 

Este proyecto es una demostración práctica de Angular Material , la biblioteca de componentes de interfaz de usuario (UI) de Google diseñada para crear aplicaciones web modernas y accesibles. Utiliza 15 componentes principales de Angular Material, cada uno configurado y estilizado para ofrecer una experiencia de usuario fluida y profesional.

Propósito del Proyecto 

El objetivo principal de este proyecto es mostrar cómo integrar y utilizar los componentes de Angular Material en una aplicación real. Cada componente está implementado de manera sencilla pero efectiva, permitiendo que los usuarios experimenten su funcionalidad y apariencia. 

Este proyecto también sirve como referencia para desarrolladores que deseen aprender más sobre Angular Material y cómo aplicarlo en sus propios proyectos.

Componentes Implementados 

Este proyecto utiliza 15 componentes principales  de Angular Material, todos ellos disponibles en la documentación oficial: https://material.angular.dev/components/form-field/overview  . 

Los componentes implementados son: 

    Card 
    Autocomplete 
    Datepicker 
    Divider 
    Expansion Panel 
    Form Field 
    Input 
    Menu 
    Paginator 
    Progress Bar 
    Radio Button 
    Slide Toggle 
    Sorting 
    Spinner 
    Toolbar 
     

Cada componente está configurado con ejemplos básicos y personalizados para demostrar su funcionalidad.
